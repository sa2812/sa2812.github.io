Thermodynamic simulations using the Ising model

Abstract

The Ising model is a simplified model of the spin-spin interactions between neighbouring atoms which gives rise to properties of the material such as specific heat capacity and magnetic susceptibility. The Metropolis algorithm is used to model the behaviour of a square array of atoms as the temperature changes. The phase transition is observed at a value of β ≈ 0.4. In subsequent run throughs, the value of β is held constant whilst the external magnetic field is varied.
